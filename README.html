Blog engine. Combination of
Ghost Markdown Editor and
MiniBlog projects.

Language is set in web.config. 

<appSettings> 
    <add key="blog:lang" value="ru" /> 
</appSettings>

If this settings is not in web.config, language can be changed with gui and is persisted with cookie.
Server side uses MarkdownDeep, ImageResizer.
The core of client side is Ghost editor. ALso, highlight.js,
magnific popup, pure.css, codemirror.inline-attach, pagedown-extra,
selectize underscore, path are used.

Press F2 to begin editing current post.
Keyboard shortcuts in editing mode:

  • F1 � editing help
  • Ctrl+S � save
  • Ctrl+Enter � save and exit editing mode
